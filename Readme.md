# ESP8266-RTOS_Template

FreeRTOS template for the ESP8266 using PlatformIO.  
Made with the WEMOS D1 Mini in mind for quick IoT starts.

## What's included
- Basic blink task
- Basic WiFi connection task
- Simple structure to add more FreeRTOS tasks

## Requirements
- PlatformIO
- WEMOS D1 Mini (or other ESP8266 board)

## Getting started
1. Open the project in VS Code + PlatformIO
2. Build and upload

## Notes
ESP8266 RAM is tight — keep task stacks reasonable and add features gradually.
